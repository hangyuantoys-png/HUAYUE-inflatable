# Huayue Toys Next.js Website

Professional B2B foreign trade website for Guangzhou Huayue Inflatable Products Co., Ltd.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Implemented Scope

- Next.js App Router site structure
- Homepage for commercial inflatable water park positioning
- Product center and product detail templates
- Application pages for resort, rental and beach/lake buyers
- About, Factory, Download, Blog and Contact pages
- UAE, Saudi Arabia, Nigeria and South Africa SEO country pages
- FAQ, Product, Breadcrumb and Organization structured data
- `sitemap.xml` and `robots.txt`
- Client-side inquiry form success state
